Путь до журнала логирования прописываеся в: nLog.config -> <target...

Аргументы командной строки:
-p (путь до файла с данными - совпадает с каталогом программы-генератором)
-o (директория генерации - совпадает с пунктом "Директория каталога программы")
-n (имя программы - по-стандарту - main_gen - может изменяться)

Для каждого типа детали своя программа генератор и итоговая программа (название json-файла соответстует итоговой программе)

Стандартное время отклика - 10 с. // При открытии может это время грузится.

Стандартный пин-код: 123

Загружает сначала dat, потом  src, main.src в самом конце

Для каждого робота также отдельно реализованы: отправка сгенерированной программы на контрорллер и выбор, выбор сгенерированной программы (по имени главной программы) + управление программой

--------------------------Параметры--------------------------
-------------------------------------------------------------
Параметры по-умолчанию для роботов: App.config -> <robot...

ControlerFolder - путь к папке с программой на контроллере

Параметры по-умолчанию для настила с ребром: App.config -> <plita...

Generator - имя программы-генератора
SumReber - количество ребер
Long - длина настила, мм.
Wight - ширина настила, мм.
Hight - высота ребра, мм.
DistanceToFirst - расстояние до осевой линии первого ребра сверху, мм.
DistanceBetween - расстояние между осевыми линиями рёбер, мм.
DistanceToStart - расстояние до торца ребра слева, мм.
DistanceToEnd - расстояние до торца ребра справа, мм.
DissolutionStart - роспуск слева, мм.
DissolutionEnd - оспуск справа, мм.
ThicknessPlita - толщина настила, мм.
ThicknessRebro - толщина ребра, мм.
SearchOffsetStart - отступ поиска в начале, мм.
SearchOffsetEnd - отступ поиска в конце, мм.
SeamsOverlap - перекрытие швов, мм.
TechOffsetSeamStart - технологический отступ начала шва, мм.
TechOffsetSeamEnd - технологический отступ конца шва, мм.
BevelToStart - скос слева, мм.
BevelToEnd - скос справа, мм.
WildingSpead - скорость сварки, см/м
ProgramNom - номер сварачной программы
